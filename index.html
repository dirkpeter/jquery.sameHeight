Config

delay
default: 10 [ms] buffering while resizing.
initialDelay
default: 10 [ms] delay before the initial height-adjustment.
breakpoint
default: 500 [px] window with above witch the plugin sets a min-height. below min-height will be set to 0;
minHeight
default: 0 [px] min-height for selected elements, when above the breakpoint
